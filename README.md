Installation
-----

```bash
npm install coz-bud-remover --save
```

Usage
----

```javascript
var cozBudRemover = require('coz-bud-remover');

var remover = cozBudRemover({});

remover.remove('**/.*.bud', function (err, buds) {
    /*...*/
});
```
